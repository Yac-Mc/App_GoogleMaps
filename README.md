# App_GoogleMaps
App de mapa en angular con librería de googlemaps y sockets

## FrontEnd: Angular
Reconstruir módulos de Node
```
npm install
```
levantar aplicación
```
ng serve
```

## Backend: Express - Sockect

Reconstruir módulos de Node
```
npm install
```

Generar el DIST
```
tsc -w
```

Levantar servidor, cualquiera de estos dos comandos
```
nodemon dist/
node dist
```
